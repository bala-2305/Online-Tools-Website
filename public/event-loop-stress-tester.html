How It Works

This tool stresses your browser's event loop by executing various JavaScript operations that can block the main thread, simulating real-world performance issues.

Stress Operations:

  • Synchronous: Blocking CPU operations
  • Async: Promise chains and microtasks
  • DOM: Massive DOM manipulations
  • Memory: Large allocations and GC pressure
  • Network: Simulated async requests
  • Timers: Thousands of setTimeout calls

What It Measures:

  • Event Loop Blocking: Time spent blocking the main thread
  • UI Responsiveness: How the interface responds during stress
  • Memory Pressure: Impact on garbage collection
  • Performance Degradation: How performance degrades over time

Warning: This test can cause browser unresponsiveness. Use with caution and close other tabs.
