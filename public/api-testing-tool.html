How to Use

🔗 Enter URL: Input your API endpoint URL in the URL field

📋 Select Method: Choose HTTP method (GET, POST, PUT, DELETE)

📝 Add Headers: Include authentication tokens, content types, etc.

📄 Request Body: Add JSON data for POST/PUT requests

🚀 Send Request: Click "Send Request" to test your API

👀 View Response: Check status, headers, and response body

Features

🌐 All Methods: Support for GET, POST, PUT, DELETE, PATCH

📋 Custom Headers: Add any headers your API requires

📄 JSON Formatting: Auto-format JSON request/response bodies

⏱️ Response Time: Measure API response times

🍪 Cookies: View and manage cookies from responses

📚 History: Keep track of recent API requests
