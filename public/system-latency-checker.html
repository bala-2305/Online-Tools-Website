How It Works

This tool measures the delay in your browser's event loop, which indicates how responsive your system is to JavaScript execution.

What is Event Loop Latency?

The event loop latency measures how long it takes for scheduled callbacks (like setTimeout) to execute. Lower latency means better responsiveness.

  • 0-16ms: Excellent (60+ FPS equivalent)
  • 16-33ms: Good (30-60 FPS)
  • 33-66ms: Fair (15-30 FPS)
  • 66ms+: Poor (under 15 FPS)

Test Modes:

  • Normal: Standard latency measurement
  • Stress Test: Adds computational load
  • Background Load: Simulates typical usage

Tip: For best results, close other applications and browser tabs during testing.
