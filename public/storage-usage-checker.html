How It Works

This tool analyzes your browser's local storage and session storage usage, providing detailed information about stored data and available space.

Storage Types:

  • localStorage: Persistent storage that survives browser restarts
  • sessionStorage: Temporary storage cleared when tab is closed

Typical quota limits range from 5-10MB per origin, but can vary by browser.

Features:

  • Real-time Analysis: Get instant storage usage information
  • Item Breakdown: See individual stored items and their sizes
  • Export Data: Download storage contents as JSON
  • Clear Storage: Remove all stored data with one click

Warning: Clearing storage will remove all locally stored data for this site.
