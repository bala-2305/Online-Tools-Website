How It Works

This tool simulates sustained computational load to test for thermal throttling, where CPU performance is reduced to prevent overheating.

What is Thermal Throttling?

Thermal throttling occurs when a CPU reduces its clock speed and performance to stay within safe temperature limits, preventing hardware damage.

  • Performance Drop: Sudden decrease in operations per second
  • Temperature Response: CPU protecting itself from overheating
  • Recovery: Performance may return to normal after cooling

Test Modes:

  • CPU Intensive: Pure computational load
  • Memory Intensive: Memory allocation and access patterns
  • Mixed Load: Combination of CPU and memory operations

Warning: This test generates significant heat and may cause throttling. Monitor your system temperature.
